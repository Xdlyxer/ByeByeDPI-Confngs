# ByeDPI Anti-RKN Config

**Никакой цензуры. Никаких блокировок.**

Этот конфиг помогает обходить дурацкие блокировки РКН.

## Что я им обхожу
- Дискорд (чтобы общаться с бро)
- Телеграм (ну это база)(даже подходит для BBD X TG)
- Ютуб (смотреть видосики)

## Как это использовать (по-быстрому)
1. Качаешь [BBDPI](https://github.com/romanvht/ByeByeDPI)
2. Загружаешь мой файлик "BBD_AntiRKN.json"
3. Импортируешь в настройках
4. Включаешь и забываешь
