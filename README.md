# ByeByeDPI-Configs

## Что я им обхожу
- Дискорд (чтобы общаться с бро)

## Как использовать (по-быстрому)
1. Качаешь [ByeByeDPI](https://github.com/romanvht/ByeByeDPI)
2. Загружаешь мой файлик `BBD_AntiRKN.json`
3. Импортируешь в настройках
4. Включаешь и забываешь

## Доступные конфиги
- `bbd_AntiRKN.json` — базовый
- `BBD_Every_day.json` — универсальный
- `BBD_Games.json` — скоро




## 📖 Как пользоваться моими конфигами

### 1. Скачай ByeByeDPI
- Официальный Репозиторий: [ByeByeDPI](https://github.com/romanvht/ByeByeDPI)
- Скачай последнюю версию APK и установи на телефон
 **Важно!**
Выбирайте Версии Тщательно! А то могут быть неприятности:
- Для телевизоров это вторая версия начиная сверху.
- Для старых телефонов третья версия начиная сверху.


## 2. Скачай нужный конфиг
- В этом репозитории выбери файл `.json`:
  - `bbd_AntiRKN.json` — базовый, для сайтов
  - `BBD_Every_day.json` — универсальный, на каждый день
  - `bbd_Games.json` — **игровой** (Discord, YouTube, Steam, Xbox, Epic, PUBG, Among Us, Roblox)
- Нажми на файл, затем на **Download** (или «Скачать»)

### 3. Импортируй в ByeByeDPI
1. Открой приложение ByeByeDPI
2. Нажми на шестерёнку (⚙️) в правом верхнем углу
3. Нажми на три точки (⋮) → выбери **«Импорт настроек»**
4. Выбери скачанный `.json` файл

### 4. Включи обход
- Вернись на главный экран
- Нажми **«Запустить»**

### Готово! Теперь Discord, YouTube, игры и сайты должны работать без блокировок.
